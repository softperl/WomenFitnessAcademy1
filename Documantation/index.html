A) Configuring API - top

Required Plugins

We are using default WordPress Rest API. WordPress Rest API is included in WordPress core files so you don’t need to install it on your blog. You will need to install and activate JWT Authentication for WP-API For user Authentication.

Configuring “JWT Authentication for WP-API” Plugin

Most of the shared hosting has disabled the HTTP Authorization Header by default. To enable this option you’ll need to edit your .htaccess file. “.htaccess” file is located in your website’s root folder.

Please add the following code to your ".htaccess" file

RewriteEngine on
RewriteCond %{HTTP:Authorization} ^(.*)
RewriteRule ^(.*) - [E=HTTP_AUTHORIZATION:%1]

The JWT needs a secret key to sign the token this secret key must be unique and never revealed. To add the secret key edit your wp-config.php file and add a new constant called JWT_AUTH_SECRET_KEY

“wp-config.php” file is located in your website’s root folder.

define('JWT_AUTH_SECRET_KEY', 'your-top-secrect-key');

You can use a string from here

For example

define('JWT_AUTH_SECRET_KEY', '`K*6l,! DA-pP@En/5g-E-{&~QQD`eJ(g^sPO96~3(Z8@_H|7hC&H%wIl31Wr%/P');

Please note some security plugins like iThemes Security does block WordPress API after installation. To check if your API is working please enter below URL in your browsers address bar and see if it will return the content of your website. http://yourwebsite.com/wp-json/wp/v2/posts (Don’t forget to change [yourwebsite.com] with your actual domain name.

If API is not working, please follow these steps to make it available:

1: To make API available, navigate to the WordPress Tweaks section on the Security > Settings page of your WordPress dashboard. Click the “Configure Settings” button.

2. Scroll to the REST API section. Select the “Default Access” setting.

3. Click “Save Settings” to save your new settings.

B) Post Formats - top

Post Formats and Shortcodes

Currently project supports 3 post formats (Standard, Gallery, Video). If any of these formats is chosen for a Wordpress post, the relevant icon will be shown for the post within the app.

Standard Post Format Default WordPress post format. When you create a post in WordPress its automatically assigned as “Standard Format”. Please note because many WordPress sites are using Visual composer or other page builders to create posts, we have stripped all shortcodes from the content for compatibility. If you want to show images or videos within your posts, please create gallery or video shortcodes and add them into “Excerpt”.

Gallery Post Format: To show a galley post format, choose gallery format for that post and paste your gallery shortcode in post Excerpts. An example Gallery shortcode is like 

[gallery ids="2861,2860,2859,2854,2853,2852"]
. Ids are unique numbers associated with image files uploaded to your WordPress website. You can separate ids with comma ( , ).

You can use WordPress default built-in feature to create a Gallery shortcode. Don’t forget to paste your shortcode in “Excerpts”. App will show the gallery only if shortcode is pasted in Excerpts.

Video Post Format: To show a video post format, choose video format for that post and paste your video shortcode in post Excerpts. An example Video shortcode is like 

[youtube ids=Kdg4DLAPC4A,A6j7mUxGz20,USIao3W2W1E]
. ids are unique video IDs of youtube videos. For example in this Youtube URL https://www.youtube.com/watch?v=ysBN9NHOI0c&t, the last part of the URL (ysBN9NHOI0c&t) after = is the video ID. You can separate ids with comma ( , ).

E) Navigation - top

WP Connect has a very flexible navigation system where you can update your navigation items locally in your project or with a remote json file that you can upload on your web server.

If you'd like to use a local conﬁguration ﬁle, you need to edit the navigation.json file in app's assets folder. This file is located in app/assets/navigation.json

You can load navigation.json ﬁle from the web, which has the advantage that you can update your app's content dynamically, without users having to update their app. Upload your navigation.json ﬁle to your web server and make sure that its has valid reachable URL . For example: https://yourwebsite.com/navigation.json

Providers

WP Connect shows content from (Wordpress, Youtube, Webview, Facebook, Instagram). Provider value defines the source of content.

Example: if you want to show recent posts from Wordpress website, set wordpress_recent as your provider. 

"provider": "wordpress_recent",

wordpress_recent: Most recent posts from your blog.
wordpress_categories: A list of all categories on your blog.
wordpress_category: A single category from your blog.

Example: If you want to show recent videos from Wordpress website, set youtube_recent as provider value. 

"provider": "youtube_recent",

youtube_recent: Most recent videos from Youtube channel.
youtube_playlists: A list of all the playlists from Youtube channel.
youtube_playlist: A single playlist from Youtube channel.

Example: If you want to show a web page, set webview as provider value. 

"provider": "webview",
webview: Any web address (URL)

Example: If you want to show recent posts from facebook, set facebook as your provider. 

"provider": "facebook",

instagram: Your Instagram feed.
facebook: Your facebook feed.

H) Notifications - top

You can send notifications to users of your app with Firebase. To do so you need to create a Firebase project. To learn how to create your firebase project read this.

New Firebase project will generate a file called "google-services.json". Download and copy this file to app\ directory in your project so you will have app\google-services.json.

Now you can go to Firebase console and push a notification to your app, see reports and analytics and get crash reports.

Follow the below steps to change your package name:

  • Right click on the root folder of your project.
  • Click “Open Module Setting”.
  • Select the module ‘App’ if it is not already selected.
  • Go to the ‘Flavours’ tab.4. Go to the ‘Flavours’ tab.
  • Change the applicationID to whatever package name you want.
  • Press OK.
